# Svelte action set-focus

- will set focus on `a` or `button` as soon as element mounts.
- will return focus to previous element when closed

## How to
```html
<div
    use:setFocus>
    <h2>I am an overlay</h2>
    <button>Focus will be set here</button>
</div>
```
